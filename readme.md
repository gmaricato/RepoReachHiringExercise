## Get started

### Install dependencies
```bash
$ yarn install
```

### Run development server
```bash
$ yarn start
```

### Build and run
```bash
# For iOS
$ npx pod install
$ ./run-ios.sh

# For Android
$ yarn android
```

### Run tests
```bash
$ yarn test
