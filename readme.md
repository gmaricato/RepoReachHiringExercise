# RepReach Hiring Exercise

### Introduction
* An app where you can search for a list of doctors by name, procedure and scheduling availability

## Get started

### Install dependencies
```bash
$ yarn install
```

### Run development server
```bash
$ yarn start
```

### Build and run
```bash
# For iOS
$ npx pod install
$ ./run-ios.sh

# For Android
$ yarn android
```

### Run tests
```bash
$ yarn test
